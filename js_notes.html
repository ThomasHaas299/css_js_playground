Im CSS ist ein schönes Beispiel für den Schachtelungs-Selektor zu sehen.

String

let str = "ABCDEFGabcdefg0123467";
str.length;
str.charAt(2);

Seit ES 2022 gibt es auch str.at(). Das Ergebnis ist das Selbe wie bei str.charAt(), aber es werden auch negative Argumente akzeptiert und verarbeitet:

str.at(2);
str.at(-1);

Array

let arr = ["A", "B", "C", "a", "b", "c", "0", "1", "2", ];
typeof arr;
arr.length;
Array.isArray(arr);

Analog zu String.at() gibt es auch Array.at(), mit der Möglichkeit eines negativen Arguments.

arr.at(2);
arr.at(-1);

Ein Array kann sortiert werden, auch mit einer Sortierfunktion und damit auch zufällig:

arr.sort();
arr.sort((a, c) => Math.random() - 0.5);
